# iieg_dbt
Un ejemplo para crear un pipeline en dbt para IIEG.

## Prerequisitos
### [uv](https://docs.astral.sh/uv/)
- `brew install uv`
- `powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"`

## Como ejecutar
Seguir a los cells del jupyter notebook con un kernel establecido por el `venv` de uv--se puede crear a los files sql manualmente o automaticamente por el notebook para una creacion automizada.
